Configuring your linux desktop using conky

This blog is about configuring your linux desktop using conky.

After I configured my desktop it looked like this.

So I hope that you too want something like this.

First you need to install conky in your computer. Go to your terminal and type

sudo apt-get install conky conky-all

Next you should install some extra features like sensors and temperature measurement. For that type

sudo apt-get install curl lm-sensors hddtemp

Now that conky is installed go to your home folder and search for the folder .conky If not there,create a new folder and name it .conky

Now you need to store all your conky files in this folder.Create as many conky files as you want and store them here

Name your conky files as .conkyrc_name . For example I have created 3 conky files namely .conkyrc_weather , .conkyrc_me and .conkyrc_clock which display weather data,computer data and clock respectively.

You can view my conky files in my github repo here.

Next you start viewing your customised desktop by typing in tour terminal

conky -c ~/.conky/.conkyrc_name

This will only show a single conky file named .conkyrc_name

For viewing multiple conky in your desktop you need to edit your .bash_aliases file in your home directory.Edit it as follows- At the bottom of the file add

alias myconky="conky -c ~/.conky/.conkyrc_name1 & conky -c ~/.conky/.conkyrc_name2 & conky -c ~/.conky/.conkyrc_name3"

Where name1,name2,etc are the name of your conky files.

Now you have set all your conky files to start when you type myconky in your terminal.

Now go forward.Edit your conky files as you want your desktop to be seen.Have fun.
